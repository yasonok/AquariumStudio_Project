## 文章規範（所有文章通用）

### 內容要求
- ✅ 每篇 3000+ 字
- ✅ 6 張本地圖片（圖文穿插，分散在各段落）
- ✅ **顧問穿插**（顧問觀點、案例分享、比較表格、顧問建議）
- ✅ SEO 優化（標籤、關鍵字、描述）

### 顧問穿插的內容類型
1. **顧問觀點** 💡 - 用第一人稱分享專業見解
2. **顧問提醒** ⚠️ - 重點提示或常見錯誤
3. **顧問經驗** 👤 - 個人實測經驗或案例
4. **比較表格** 📊 - 工具/方案比較
5. **顧問建議** 🎯 - 具體行動建議

### 圖片處理規範（重要！）

**未來發布新文章的圖片處理流程：**
1. ✅ 從 Unsplash 下載圖片到 `source/images/`
2. ✅ **驗證圖片大小**：`ls -la *.jpg | awk '$5 < 10000 {print "損壞: " $9}'`
3. ✅ 刪除損壞圖片（29 bytes 是錯誤頁面）
4. ✅ 從 Picsum 重新下載：`curl -sL "https://picsum.photos/800/600?random=N" -o photo-XXXXX.jpg`
5. ✅ 將 Markdown 中的 URL 替換為本地路徑 `/images/photo-xxxx.jpg`
6. ✅ 重新部署

**這樣所有圖片都會隨網站部署，不會有外部載入問題！**
